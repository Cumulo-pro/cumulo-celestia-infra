# Research

Ecosystem research and technical evaluations by Cumulo, covering projects and infrastructure relevant to the Celestia ecosystem. Each subfolder corresponds to a single project or topic, and contains a self-contained report with supporting evidence (screenshots, transaction hashes, data) where applicable.

## Reports

- [`bot-fun/`](./bot-fun/) — Technical evaluation of [bot.fun](https://bot.fun), an AI-agent memecoin trading platform running on Eden (Celestia's native EVM chain). Combines a hands-on controlled test with corroborated public feedback from bot.fun's Discord community.
